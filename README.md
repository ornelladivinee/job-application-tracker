# 📊 Job Application Tracker

A Python-based tool to track and manage internship/job applications across Europe.

## 🚀 About the project

This project helps organize internship applications in a structured way using Python and Excel.  
It was built to improve tracking, follow-ups, and personal organization during the job search process.

The goal is to have a clear overview of applications across different companies and countries.

## 🛠️ Technologies used

- Python 🐍
- Pandas 📊
- Excel (XLSX)

## 📁 Project structure

- `tracker.py` → creates the Excel tracking file  
- `add_application.py` → adds new job applications automatically  
- `applications_tracker.xlsx` → stored applications data  

## 🌍 Example companies applied to

- Siemens (Portugal)
- Airbus (Spain & Portugal)
- Capgemini (Spain)
- NTT Data (Spain)

## 🎯 Features

- Add job applications automatically
- Track status (Applied, Interview, Offer, Rejected)
- Centralized Excel database
- Easy follow-up tracking
